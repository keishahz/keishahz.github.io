## Deskripsi Project

Membangun aplikasi web interaktif menggunakan Streamlit untuk menganalisis dan merekomendasikan film berdasarkan data IMDb.

### Fitur Utama

* **Visualisasi Tren**: Melihat tren film dari masa ke masa.
* **Filtering Dinamis**: Menyaring film berdasarkan kriteria tertentu.
* **Rekomendasi Berbasis Mood**: Memberikan saran film sesuai suasana hati pengguna.

### Teknologi yang Digunakan

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Streamlit untuk UI interaktif
- IMDb Dataset
