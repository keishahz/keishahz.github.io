## Deskripsi Proyek

Proyek akhir program **SISTECH 2025** yang mencakup seluruh siklus data science.

### Tahapan Pengerjaan

1. **Data Acquisition**: Melakukan web scraping data kursus dari Class Central menggunakan **Selenium**.
2. **Preprocessing**: Pembersihan teks (Stemming & Stopwords Removal) menggunakan **NLTK**.
3. **Modeling**: Membangun sistem rekomendasi Content-Based Filtering menggunakan **TF-IDF** dan **Cosine Similarity**.
